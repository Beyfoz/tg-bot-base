# aiogram 3.x Telegram Bot Base

Готовый структурированный шаблон для сборки коммерческих Telegram-ботов (визитки, сбор заявок, анкетирование). Настроен под деплой на Linux-сервер.

## Что внутри
* **Разделение логики:** Архитектура на роутерах (`aiogram.Router`), хэндлеры и конфиги разнесены по отдельным модулям.
* **FSM:** Настроен пошаговый сбор данных от пользователя (анкеты/заявки) через `FSMContext`.
* **Clean UI:** Меню и переключение шагов сделаны через отправку новых сообщений — это сохраняет историю переписки и не ломает интерфейс.
* **Эхо-фильтр:** Пользователь не сломает логику случайным текстом.

## Стек
* Python 3.12+
* aiogram 3.x
* Полная совместимость: Windows (разработка) / Ubuntu Server (деплой)
